# My stars

Simple cli script to show you your n last stars on github. 

## Usage

        Usage: my-latest-stars [options]

        Options:
          -h, --help   Show help
          -l, --limit  Limit of the stars to show, default is 10. 0 = no
                       limit.

        Made by DKunin http://dkunin.github.io/
